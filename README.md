# Getting started 

### Installing
```
npm install    
webpack --watch
```

### Shortcodes

| Shortcode | Attribute name | Default value | Attribute type |
| --------- | -------------- | ------------- | -------------- |
| person    | id             | ''            | named          |
| person    | flag           | ''            | numeric        |

### Blocks

| Block            | Attribute name | Default value | Data type |
| ---------------- | -------------- | ------------- | --------- |
| namespace/person | id             | ''            | string    |
| namespace/person | flag           | ''            | string    |
